## How to use

### Loading a ROM

#### Through CLI

`cargo run <path_to_rom>`

#### Dragging a file

Drag and drop a ROM onto the window once running

### Default keys

#### Chip8 keypad mappings

| 0        | 0     |
| 1        | 1     |
| 2        | 2     |
| 3        | 3     |

#### Other

| Keyboard | Action             |
|----------|--------------------|
| p        | Toggle execution   |
| n        | Step 1 instruction |

## Dependencies

### Linux

`sudo apt-get install libsdl2-dev`

### Mac

`brew install sdl2`
